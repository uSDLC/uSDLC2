Project

uSDLC is an open source, platform independent application for use by all members of a software development project. It provides direct connections between vision, design and the production code. There are processes to support, but not enforce, the use of behaviour driver development. It supports living lean documentation practices.

uSDLC helps a software project be successful, more economical with a more reliable result without adding load to the members of the team.
