Install uSDLC

  1. Go to http://usdlc.net/install
  2. Select where to download the installation script
  3. Run the script
  4. Tell it where you want uSDLC installed
  5. Watch the reported downloads
    1. uSDLC2
    2. Node-js
    3. Node modules
  6. uSDLC2 will start and open a browser to the home screen
