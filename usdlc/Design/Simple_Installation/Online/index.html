Install uSDLC

  1. Go to https://github.com/uSDLC/uSDLC2
  2. click download link in README.md for your OS
  3. create a directory uSDLC2 in your dev folder and download to it.
  4. Run the script
  5. Watch the reported downloads
    1. uSDLC2
    2. Node-js
    3. Node modules
  7. uSDLC2 will start and open a browser to the home screen
