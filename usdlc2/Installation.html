Installation

Single Script Install

Just download and run https://raw.github.com/uSDLC/uSDLC2/master/install-usdlc2.sh or https://raw.github.com/uSDLC/uSDLC2/master/install-usdlc2.sh.bat. The former will work on Unix based systems (including OS X).

For Linux or OS X, go to the target directory and use the following script (or just copy and paste the contents to a shell session):

Run the uSDLC2 server and it will download roaster dependencies. You don't need to be in the directory, so just specify the command on your startup to have uSDLC2 available at all times.

In a browser, enter http://localhost:9009 and an additional set of uSDLC2 client dependencies will download. Future page loads will be fast, but this one can take 40 seconds on ADSL.

Update the installation by re-running the script at any time. Changes will not be lost.

Ref: install-usdlc2.shgo.sh

Command Line Options

A server is run with go.sh server options... where options are name=value pairs.

config=debug

Configuration can be set on the command line or in a configuration file at uSDLC2/config. Debug mode

  • Display request on server console
  • Reduces browser caching to 10 seconds
  • Sets up node debugging for the server
  • Adds a server restart link to every page
Ref: debug.coffee

port=9009

Set the http port used to access uSDLC2.

User Management

Change records and pairing both require an understanding of the current user. Set your user once using the uSDLC button of that name (or Alt-U).
