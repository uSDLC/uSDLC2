Metadata

Every section can have different forms of metadata attached. Like footnotes they are visible at the end of the section with distinct formatting. Like any other element they are available for editing. Typically they are created as a byproduct of another operation.

References

Reference metadata is a list of links to information of interest in the current section.

Source Editing

If you choose to edit any production source files from a section a link is kept so that they are easier to find next time.

Bridge

When a Given/When/Then (GWT) element is created in a section a link is created to the instrumentation code contained within the document.

Play GWT

When a Given/When/Then (GWT) element is created in a section a link is created to execute (play) the instrumentation code connecting the documentation to the production code.

Other Pages and Sections

The editor context menu allows links to other pages or sections to be added for cross-referencing.
